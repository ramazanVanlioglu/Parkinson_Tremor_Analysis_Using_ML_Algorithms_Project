# Parkinson_Tremor_Analysis_Using_ML_Algorithms_Project
The project is generally focuses on patients that have Parkinson disease. First, the prototype device scales the tremor and analysis it's tremor level with machine learning algorithms.  As an interface, the Blynk app is used. The level of tremor and the models possibility of inference displays on the Blynk app.
